# 📊 Student Performance Analytics System

## Overview

This project analyzes student academic performance using Python and data analytics techniques. The goal is to identify factors that affect student success and generate meaningful insights through data visualization and analysis.

## Objectives

* Clean and preprocess student data
* Analyze relationships between study habits and academic performance
* Identify students at academic risk
* Create visual reports and insights
* Build an interactive dashboard for data exploration

## Dataset Features

* Study Hours
* Internal Marks
* Final Marks
* Assignment Scores
* Attendance
* Family Income
* Internet Access
* Extra Activities
* Placement Status

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Streamlit
* Google Colab

## Project Workflow

### 1. Data Cleaning

* Removed duplicate records
* Handled missing values
* Standardized data formats
* Prepared data for analysis

### 2. Exploratory Data Analysis (EDA)

* Study Hours vs Final Marks
* Attendance vs Performance
* Internet Access vs Performance
* Correlation Analysis
* Distribution and Outlier Detection

### 3. Feature Engineering

* Risk Category
* Performance Level
* Additional analytical features

### 4. Dashboard Development

* Interactive student performance dashboard
* Key performance indicators (KPIs)
* Data filtering options
* Visual analytics and insights

## Key Insights

* Students who study more tend to achieve higher marks.
* Higher attendance is associated with better academic performance.
* Internet access positively impacts learning outcomes.
* Early identification of at-risk students can improve academic support.
